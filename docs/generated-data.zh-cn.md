# 生成数据

AsmKit 目标数据来自 LLVM target records。仓库中提交的生成文件由项目修改后的
TableGen-based 生成器 `asmkit-tblgen` 生成。

构建 AsmKit 不会运行 LLVM、Python、C++ 或 `asmkit-tblgen`。生成的 C 和头文件
是静态库的普通源码输入。

## 生成路径

生成的 AsmKit 表位于：

- [`src/generated/`](../src/generated/)

生成的公开目标 ID 位于：

- [`include/asmkit/target/`](../include/asmkit/target/)

生成文件包含 provenance 注释和 SPDX 标记：

```text
Generated by asmkit-tblgen from LLVM llvmorg-22.1.7 target records.
SPDX-License-Identifier: Apache-2.0 WITH LLVM-exception
```

## 生成内容

生成数据包括：

- Decode 表。
- 结构化 encode 表。
- 指令元数据。
- 操作数 schema。
- 寄存器元数据。
- 谓词元数据。
- 特性元数据。
- CPU model 元数据。
- 助记符文本表。
- BPF opcode-field literal 元数据。

## 修改后的 TableGen 管线

生成管线消费 LLVM target records，并输出 AsmKit 专用 C 数据。

AsmKit overlay 补充 LLVM target records 中不能完整表达的行为，例如
PC-relative fixup、branch-stub 选择、detour 规划和目标特定 relocation rejection。

## 许可证边界

LLVM 派生生成文件使用 `Apache-2.0 WITH LLVM-exception`。项目自有源码和文档
文件使用 MIT。见 [`license.zh-cn.md`](license.zh-cn.md)。
